# AED-Map-Project
# AED(자동심장충격기) 의무 설치 현황 분석

> 공공데이터를 활용하여 서울시 AED(자동심장충격기) 설치 현황을 분석하고,
> 설치가 필요한 지역을 데이터 전처리 및 시각화를 통해 도출한 팀 프로젝트입니다.

## 📅 프로젝트 정보
- **기간** : 2023.02.01 ~ 2023.02.03
- **구성** : 팀 프로젝트 (5인)
- **개발 환경** : Python 3, Jupyter Notebook

## 👥 팀원
- 강지인
- 김유림
- 김호영
- 이선주
- 황도희

## 🛠 Tech Stack

- Python
- Pandas
- Matplotlib
- Seaborn
- Folium
- ipyleaflet

## 📌 프로젝트 목표

- 서울시 AED 설치 현황 분석
- 공공장소 AED 설치 여부 확인
- 데이터 전처리 및 지도 시각화를 통한 설치 필요 지역 도출
- AED 교육 및 인식 현황 분석

## 📂 프로젝트 구성

```
├── data/          # 공공데이터
├── notebook/      # Jupyter Notebook
├── result/        # 결과 이미지 및 지도
└── README.md
```

## 💡 주요 기능

- 서울시 AED 설치 위치 지도 시각화
- 주민센터 AED 설치 여부 확인
- 서울 지하철역 AED 설치 여부 확인
- 지역별 AED 위치 조회
- AED 인식 및 교육 현황 시각화

## 🙋 담당 역할 (김유림)

- 서울시 AED 위치 데이터 전처리
- Folium을 활용한 AED 지도 시각화
- 원하는 서울 지역 입력 시 AED 위치 조회 기능 구현
- 공공데이터 조사 및 정리

## 📊 프로젝트 결과

- AED 교육 필요성과 설치 확대 필요성 도출
